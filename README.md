# Runner game

This project is a runner game. Technologies: [PixiJS](https://pixijs.com/), [TypeScript](https://www.typescriptlang.org/) and [Webpack](https://webpack.js.org/)

To run the package.

```
git clone https://github.com/aliaksandrminsk/mvc-runner-game
cd mvc-runner-game
npm install
npm run start
```
